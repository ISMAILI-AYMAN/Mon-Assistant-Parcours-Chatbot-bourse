===============================
MON ASSISTANT PARCOURS BOURSE
===============================

Bienvenue dans la documentation officielle de **MON ASSISTANT PARCOURS CHATBOT BOURSE**.

Ce chatbot est conçu pour guider les étudiants désireux de bénificier d'une bourse d'étude en donnant les informations sur les bourses disponibles tel que:
-les conditions nécessaires pour candidater 
-les documents à fournir
-Suggestions de bourses en fonctions des informations personnelles
